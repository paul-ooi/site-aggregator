Site Aggregator is a lightweight content aggregation system that collects and displays articles from web sources and RSS feeds.

## Technology Stack

- **Framework**: [Nuxt.js 3](https://nuxt.com)
- **Language**: [TypeScript](https://www.typescriptlang.org)
- **Styling**: [SCSS](https://sass-lang.com)
- **Build Tool**: [Vite](https://vite.dev)
- **Hosting**: [GitHub Pages](https://pages.github.com)

## Content Collection

Articles are aggregated from RSS feeds and web sources using automated collection pipelines. Content is stored as markdown files and rendered at build time.

## Privacy

This site does not use cookies, analytics, or tracking scripts. No personal data is collected or stored.

## Source Code

This project is open source and available on [GitHub](https://github.com).
